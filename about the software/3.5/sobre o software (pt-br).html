Que software é esse?

> este software é baseado em Python 3.10.1 e é uma automação portátil para Smart2Sign, o mesmo é responsável por lançar o software, monitorar, gerenciar e registrar atividades do mesmo.


O que é necessário para executar esse software?

> para executá-lo você deve ter o software Google Chrome instalado.


Como configurar esse software?

- Para isso abra o arquivo ( settings.json ) e altere as propriedades abaixo.

  • "enabled" habilita/desabilita um monitor específico

  • "coordsx" posiciona a aplicação no eixo X

  • "coordsy" posiciona a aplicação no eixo Y

  • "DIR" informa o local onde os dados do monitor específico serão salvos
