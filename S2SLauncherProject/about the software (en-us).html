What software is this?

> this software is based on Python 3.10.1 and it's a portable automation for Smart2Sign, he is responsible for launching the software, monitoring, managing and recording its activities.


What is needed to run this software?

> to run it you must have the software Google Chrome installed.


How to configure this software?

- To do this open the file ( settings.json ) and change the properties below.

  • "enabled" enable/disable a specific monitor

  • "coordsx" positions the application on the X axis

  • "coordsy" positions the application on the Y axis

  • "DIR" tells you where the data for the specific monitor will be saved
